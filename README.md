# Calculator Project

這是一個使用 **Delphi VCL** 開發的基本計算機專案，採用 **MVC 架構** 分離畫面、控制與運算邏輯，讓程式結構更清楚，也更容易維護。

## 功能

此計算機支援以下功能：

- 基本四則運算：加、減、乘、除
- 小數點輸入
- 等號計算
- 清除全部（C）
- 清除目前輸入（CE）
- 退格刪除（Backspace）
- 正負號切換（±）
- 百分比運算（%）
- 平方根（Sqrt）
- 平方（Sqr）
- 倒數（1/x）
- 運算歷史紀錄顯示

## 專案架構

本專案主要分為三個部分：

### 1. View（畫面層）
負責使用者介面與按鈕事件處理，主要在 `Calculator.pas` 中。

### 2. Controller（控制層）
負責接收畫面操作，並控制資料流與運算流程，主要在 `CalculatorController.pas` 中。

### 3. Model（模型層）
負責實際的數學運算與狀態管理，主要在 `CalculatorEngine.pas` 中。

---

## 檔案說明

### `Calculator.pas`
主視窗 Form，負責：
- 顯示結果
- 接收按鈕事件
- 顯示歷史紀錄
- 與 Controller 互動

### `CalculatorController.pas`
控制器，負責：
- 處理數字按鍵輸入
- 處理運算子按鍵
- 處理等號與單元運算
- 控制畫面顯示更新
- 錯誤處理

### `CalculatorEngine.pas`
運算核心，負責：
- 儲存目前運算狀態
- 執行加減乘除
- 處理平方根、平方、倒數、百分比
- 管理是否為新輸入數字

## 主要設計概念

### MVC 架構
本專案將功能分開處理：

- **View**：只負責介面顯示
- **Controller**：負責接收事件並協調流程
- **Model**：只負責數學邏輯

這種設計可以降低程式耦合度，也讓之後新增功能更方便。

### 介面設計
`ICalculatorView` 介面用來讓 Controller 與 View 之間溝通，避免 Controller 直接依賴畫面元件。

## 使用方式

### 數字輸入
按下數字按鈕後，會將輸入顯示在右上方的結果欄位中。

### 基本運算
例如：

1. 輸入 `12`
2. 按下 `+`
3. 輸入 `8`
4. 按下 `=`

結果會顯示為 `20`

### 單元運算
- `Sqrt`：開根號
- `Sqr`：平方
- `1/x`：倒數
- `%`：百分比
- `±`：正負號切換

---

## 運算歷史

當視窗寬度大於 600 時，右側會顯示歷史紀錄區，會記錄像下面這樣的內容：

- `1 + 2 = 3`
- `Sqrt(9) = 3`
- `(50)% = 0.5`

---

## 錯誤處理

本專案已加入基本例外處理，例如：

- 除以 0
- 負數開根號
- 倒數為 0

當發生錯誤時，系統會跳出提示訊息，避免程式直接中斷。
